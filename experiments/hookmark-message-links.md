# Hookmark message links

Status: Discovery only — not shipped or committed.

Professionals want to link an email from a task or note and reopen it without
searching. Distribution requires Hookmark's complete campaign and a permanent
tagged guide visible before Tecotype is installed → first successful message
link → paid; otherwise this is compatibility only.

Signal: Hookmark supports RFC-identifier-based
[message links](https://hookproductivity.com/help/integration/using-hookmark-with-email-apps/)
in several mail clients; Tecotype stores RFC message identifiers.
