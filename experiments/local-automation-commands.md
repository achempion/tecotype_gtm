# Local automation commands

Status: **Parked** — exact placement, capacity, and paid bridge are unproven.

People using launchers, timers, text tools, or programmable keyboards need to
invoke email actions without brittle keystrokes or granting a cloud service
mailbox access. A command pack becomes distribution only when its owner reviews
and permanently features it with an attributable first-command-to-paid path.

Signal: Tecotype already models many interface behaviors as internal commands,
but no external automation API exists.
